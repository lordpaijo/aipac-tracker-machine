# SIMPLE ATM SIMULATION WRITTEN IN COBOL

Simple ATM simulation written in COBOL for the sole purpose of learning the language.
![ss-0](https://github.com/lordpaijo/aipac-tracker-machine/blob/master/ss-0.png)

## How to replicate

1. Make sure you have GnuCOBOL installed.
2. Clone the repo.
3. Run the `setup.sh` script.
4. Run the program binary and test it yourself.
