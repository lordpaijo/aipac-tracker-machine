# SIMPLE ATM SIMULATION WRITTEN IN COBOL

Simple ATM simulation written in COBOL for the sole purpose of learning the language.
![ss-0](https://github.com/lordpaijo/aipac-tracker-machine/blob/master/ss-0.png)

## How to replicate

1. Clone the repo.
2. Make a `data/` directory.
3. Build `src/init-accounts.cbl` for the account initializer binary.
4. Build `src/main.cbl` for the program binary.
5. Run the account initializer binary to initialize accounts.
6. Run the program binary and test it yourself.
